# ANKIT Vishwa

A next-generation interactive portfolio built as a futuristic operating system.

Instead of reading a traditional resume, recruiters can explore projects, infrastructure experience, technical skills, AI assistant, live terminal, and enterprise architecture through an immersive experience.

## Features

- Interactive OS Interface
- AI Assistant
- Live Terminal
- Experience Timeline
- Project Showcase
- Resume Download
- Recruiter Mode
- GitHub Integration
- Webhook Notifications
- Fully Responsive
